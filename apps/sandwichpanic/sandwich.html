どんなゲーム？

次々におちてくる食材を受け止めてサンドイッチを作るパニックアクションパズルゲームです。 うまくかわして早めに完成させたり、高く積んで高得点を狙ったり、うまく状況に合わせてサンドイッチを完成させよう。 サンドイッチを作って売るとお金を稼ごう、稼いだお金でインテリアを購入して部屋を賑やかにできます。

サンドイッチをたくさん作ろう

落ちてくる食材を受け止めてサンドイッチをたくさん作って売ってお金を稼ぐことが目的です。 時間が経つと食材が落ちるスピードが上がって難しくなっていきます。 また食材を落としたり、ぶつけたりするとミスになり、３回ミスするとゲーム終了です。 ミスしないで長くゲームを続けてたくさんのお金を稼ぎましょう。

ベーシックなGame AとピーキーなGame B

Game AとBの二つのルールで遊べます。 Game Aは「最大の大きさのサンドイッチを作る」「好みの具材の多く含む」でチップがもらえます。 これらの条件を満たしてチップをもらってより多くのお金を稼ぎましょう。

Game Bは「好みの具材だけできるだけ大きいでサンドイッチを作る」でチップがもらえ、ピーキーでうまくいけばGame Aより多くのお金が稼げます。 Game Bは通販Game Bのインテリアを購入すると遊べるようになります、まずはGame Aでお金を貯めましょう。

（ゲームのおまけ）ゲームで貯めたお金とリアルマネーで作者の部屋を整えよう

ゲームのおまけで作者の部屋に遊びに行けます。 最初は部屋が空っぽですがマシンを触ると貯まったお金を使って通販サイトでインテリアを買えます。 インテリアを買うと部屋に設置されどんどん賑やかになっていきます。通販の半分はゲームで貯めたお金で、もう半分はリアルなお金で購入できます。 リアルなお金は作者の生活費と次回作の開発予算に使われます（毎度ありがとうございます。）
