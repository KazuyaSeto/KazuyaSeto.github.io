ゲーム内容

次々におちてくる食材を受け止めてサンドイッチを作るパニックアクションパズル ゲームウォッチのようなシンプルゲームです。

サンドイッチをたくさん作ろう

ベーシックなGame AとピーキーなGame B

Game AとBの二つのルールで遊べます。 Game Aは「最大の大きさのサンドイッチを作る」「好みの具材の多く含む」でボーナスがもらえます。 より多くのお金を稼ぎましょう。

Game Bは「好みの具材だけできるだけ大きいでサンドイッチを作る」でボーナスがもらえ、よりピーキーでうまくいけば多くのお金がたまります。 Game Bは通販Game Bのインテリアを購入すると遊べるようになります、まずはGame Aでお金を貯めましょう。

（ゲームのおまけ）ゲームで貯めたお金とリアルマネーで作者の部屋を整えよう

ゲームのおまけで作者の部屋に遊びに行けます。 最初は部屋が空っぽですがマシンを触ると貯まったお金を使って通販サイトでインテリアを買えます。 インテリアを買うと部屋に設置されどんどん賑やかになっていきます。通販の半分はゲームで貯めたお金で、もう半分はリアルなお金で購入できます。 リアルなお金は作者の生活費と次回作の開発予算に使われます（毎度ありがとうございます。）
